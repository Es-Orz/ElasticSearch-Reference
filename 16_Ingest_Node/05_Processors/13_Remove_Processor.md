## Remove Processor

Removes an existing field. If the field doesn’t exist, an exception will be thrown.

 **Table 26. Remove Options**

Name |  Required |  Default |  Description  
---|---|---|---  
  
`field`

| 

yes

| 

-

| 

The field to be removed  
  
  

    
    
    {
      "remove": {
        "field": "foo"
      }
    }
